proxima

W elcome to proxima! Proxima is an animation library built with a minimal set of js on top of XElement.

A nimations in web have their limitations or require huge libraries with thousands of js code lines. With a very few lines proxima is different, while not a suite for every need, it will cover most of cases to create nicely animated sites.

Examples

<X
	type="h2"
	iterations="1"
	duration="600"
	delay="2000"
	trigger="visible"
